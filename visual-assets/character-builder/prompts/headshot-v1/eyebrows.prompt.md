# Headshot V1 - Eyebrows Visual Set Prompt

## Purpose

Create one source sheet for the `Face > Eyebrows` visual-card picker. The sheet represents eyebrow shape options only and must not imply eye shape, makeup, gaze, expression, skin tone, or gender.

## Source Sheet Prompt

Create a clean black-and-white illustrated eyebrow shape source sheet with exactly 7 separate eyebrow pairs arranged in one horizontal row, no labels.

Seven eyebrow shape icon pairs only:

1. straight eyebrows
2. soft arched eyebrows
3. natural thick brows
4. thin eyebrows
5. defined eyebrows
6. well-groomed eyebrows
7. natural eyebrows

Style: crisp black and dark gray line art on pure white background, simple elegant fashion and beauty UI asset style, consistent with clean illustrated face and eye option icons.

Composition: one horizontal row of 7 evenly spaced cells. Each item is centered, same scale, same baseline, with generous white padding for grid slicing.

Avoid: text, labels, numbers, watermark, skin, eyes, face, makeup color, colored fill, shadows, gradients, texture, extra facial features.

## Review Notes

This field intentionally uses the existing 7 eyebrow attributes without adding a new option. If the set needs a balanced 4x2 layout later, add an eighth real eyebrow attribute first instead of generating a visual-only placeholder.
