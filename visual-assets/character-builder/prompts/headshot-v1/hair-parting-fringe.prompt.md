# Headshot V1 - Hair Parting / Fringe Visual Set Prompt

## Purpose

Create one source sheet for the `Hair > Parting / Fringe` visual-card picker. This field describes hair parting, forehead exposure, and bangs/fringe only. Hair length, cut/style, texture, finish, and color remain separate semantic axes.

## Source Sheet Prompt

Create a clean black-and-white illustrated hair parting and fringe source sheet with exactly 6 separate head-and-hair icons arranged in a 2 row by 3 column grid, no labels.

Six front-facing simplified blank face head silhouettes with similar medium-length neutral hair, each clearly showing a different parting or fringe arrangement only:

1. center part with no bangs
2. side parting with no bangs
3. swept back hair with no visible part and no bangs
4. curtain bangs parted in the middle
5. blunt straight full bangs across the forehead
6. see-through wispy bangs

Style: crisp black and dark gray line art only on pure white background, simple elegant beauty UI asset style, clean vector-like contour, consistent line weight, no facial features, no color fill.

Composition: 2 rows by 3 columns. Each icon is centered, uses the same head size and similar shoulder or neck context, with generous white padding for grid slicing.

Avoid: text, labels, numbers, watermark, eyes, nose, mouth, skin color, makeup, colored fill, shadows, gradients, background texture, and distinct cut/style changes.

## Review Notes

This field should answer "how does the hair sit around the forehead?" It should not decide total hair length, color, texture, or cut/style.
