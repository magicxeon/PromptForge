# Headshot V1 - Hair Length Visual Set Prompt

## Purpose

Create one source sheet for the `Hair > Length` visual-card picker. The sheet represents hair length only and should avoid encoding texture, color, finish, bangs, or detailed hairstyle where possible.

## Source Sheet Prompt

Create a clean black-and-white illustrated hair length source sheet with exactly 4 separate head-and-hair silhouette icons arranged in one horizontal row, no labels.

Four front-facing simplified head silhouettes with hair length only:

1. buzz cut
2. short hair
3. long hair
4. extra long hair

Style: crisp black and dark gray line art only on pure white background, simple elegant beauty UI asset style, clean vector-like contour, consistent line weight, no facial features, no color fill.

Composition: one horizontal row of 4 evenly spaced cells. Each icon is centered, uses the same head size and baseline, and shows enough shoulder or neck context to compare hair length.

Avoid: text, labels, numbers, watermark, eyes, nose, mouth, makeup, skin color, colored fill, shadows, gradients, background texture, strong fashion styling, or color-specific cues.

## Review Notes

This field only controls length. Hair texture, cut/style, parting/fringe, finish, and color remain separate axes.
