# Headshot V1 - Lips Visual Set Prompt

## Purpose

Create one source sheet for the `Face > Lips` visual-card picker. The sheet represents neutral closed-mouth lip shape only. It must not imply smile, emotion, lipstick color, makeup intensity, teeth, skin tone, ethnicity, or gender.

## Source Sheet Prompt

Create a clean black-and-white illustrated lip shape source sheet with exactly 8 separate closed-mouth lip icons arranged in a 2 row by 4 column grid, no labels.

Eight front-facing neutral closed-mouth lip shape icons only:

1. natural lips
2. cherry lips
3. cupid's bow lips
4. plump lips
5. thin lips
6. heart-shaped lips
7. heavy upper lip
8. wide lips

Style: crisp black and dark gray line art on pure white background, simple elegant beauty UI asset style, consistent with the existing clean illustrated face, eyes, eyebrows, and nose option icons.

Composition: 2 rows by 4 columns. Each lip icon is centered, same scale, same baseline within each row, with generous white padding for grid slicing.

Avoid: text, labels, numbers, watermark, smile expression, open mouth, teeth, tongue, face, nose, skin color, makeup color, colored fill, shadows, gradients, texture, extra facial features.

## Review Notes

This field intentionally excludes `Soft smile`, `Closed-mouth smile`, `Gentle smile`, and `Neutral expression` because those are expression controls, not lip-shape controls.
