# Headshot V1 - Nose Visual Set Prompt

## Purpose

Create one source sheet for the `Face > Nose` visual-card picker. The sheet represents nose shape and bridge/tip structure only. It must not imply eye shape, mouth shape, skin tone, makeup, expression, ethnicity, or gender.

## Source Sheet Prompt

Create a clean black-and-white illustrated nose shape source sheet with exactly 6 separate nose icons arranged in one horizontal row, no labels.

Six front-facing minimal nose shape icons only:

1. small button nose
2. high nose bridge
3. delicate narrow nose
4. soft rounded tip
5. straight nose
6. natural nose

Style: crisp black and dark gray line art on pure white background, simple elegant fashion and beauty UI asset style, consistent with clean illustrated face, eyes, and eyebrow option icons.

Composition: one horizontal row of 6 evenly spaced cells. Each nose is centered, same scale, same vertical alignment, with generous white padding for grid slicing.

Avoid: text, labels, numbers, watermark, eyes, mouth, full face, skin color, makeup, colored fill, shadows, gradients, texture, extra facial features.

## Review Notes

Nose options can be subtle in icon form. Prioritize visible differences in bridge width, bridge height, tip roundness, and overall width while keeping the same front-facing camera angle.
