# Headshot V1 - Hair Cut / Style Visual Set Prompt

## Purpose

Create one source sheet for the `Hair > Cut / Style` visual-card picker. The sheet represents hairstyle/cut only. Hair length, texture, finish, parting/fringe, and color remain separate semantic axes when possible.

## Source Sheet Prompt

Create a clean black-and-white illustrated hair cut/style source sheet with exactly 16 separate head-and-hair silhouette icons arranged in a 4 row by 4 column grid, no labels.

Sixteen front-facing simplified blank face head silhouettes with distinct hairstyles only:

1. high ponytail
2. messy bun
3. French braid
4. layered hush cut
5. long loose waves
6. side-swept hair
7. wet look hair
8. wolf cut
9. crew cut
10. side part
11. undercut
12. pompadour
13. quiff
14. textured crop
15. Caesar cut
16. short curly crop

Style: crisp black and dark gray line art only on pure white background, simple elegant beauty UI asset style, clean vector-like contour, consistent line weight, no facial features, no color fill.

Composition: 4 rows by 4 columns. Each icon is centered, uses the same head size and similar shoulder or neck context, with generous white padding for grid slicing.

Avoid: text, labels, numbers, watermark, eyes, nose, mouth, skin color, makeup, colored fill, shadows, gradients, background texture, and color-specific cues.

## Review Notes

This field intentionally excludes straight, wavy, and curly hair because those belong to the hair texture axis. If a cut/style inherently implies length, keep the visible length only as much as needed to communicate the style.

Rows 1-2 are feminine/unisex first-pass styles. Rows 3-4 are masculine first-pass styles. The UI may filter this set by the selected character presentation while keeping the same manifest.
