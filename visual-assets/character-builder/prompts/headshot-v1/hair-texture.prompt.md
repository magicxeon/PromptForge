# Headshot V1 - Hair Texture Visual Set Prompt

## Purpose

Create one source sheet for the `Hair > Texture` visual-card picker. This field describes strand quality and surface behavior only. Hair length, cut/style, parting/fringe, finish, and color remain separate semantic axes.

## Source Sheet Prompt

Create a clean black-and-white illustrated hair texture source sheet with exactly 4 separate head-and-hair icons arranged in a 1 row by 4 column grid, no labels.

Four front-facing simplified blank face head silhouettes with the same medium-length neutral hairstyle silhouette, but each icon clearly shows a different hair texture only:

1. silky smooth straight fine strands
2. coarse thick dense strands with heavier texture
3. soft glossy waves with smooth flowing wave lines
4. frizzy voluminous hair with flyaways and airy textured volume

Style: crisp black and dark gray line art only on pure white background, simple elegant beauty UI asset style, clean vector-like contour, consistent line weight, no facial features, no color fill.

Composition: 1 row by 4 columns. Each icon is centered, uses the same head size and similar shoulder or neck context, with generous white padding for grid slicing.

Avoid: text, labels, numbers, watermark, eyes, nose, mouth, skin color, makeup, colored fill, shadows, gradients, background texture, and cut/style-specific cues.

## Review Notes

Texture should be communicated through strand density, wave pattern, flyaways, and line quality. Do not introduce distinct hairstyles that belong to `Hair > Cut / Style`.
