# Headshot V1 - Face Expression Visual Set Prompt

## Purpose

Create one source sheet for the `Face > Expression` visual-card picker. This field controls emotion only. It must not alter head pose, camera angle, hairstyle, anatomy, or scene.

## Source Sheet Prompt

Create a clean black-and-white illustrated generic facial expression source sheet with exactly 6 separate front-facing bald head / face-outline icons arranged in a 2 row by 3 column grid, no labels.

Six simplified gender-neutral front-facing face-outline icons with no hair, no beard, no makeup, the same oval head shape, and the same minimal ears/neck outline, each showing a different subtle facial expression only:

1. relaxed neutral micro-expression
2. soft thoughtful expression with calm eyes
3. subtle friendly closed-mouth smile
4. candid gentle laugh with slightly open happy mouth and eye crinkle
5. playful mild smirk with one mouth corner lifted
6. reflective neutral mood with gentle serious eyes

Style: crisp black and dark gray line art only on pure white background, simple elegant beauty UI asset style, clean vector-like contour, consistent line weight.

Composition: 2 rows by 3 columns. Each icon is centered, uses the same head size, same straight front-facing head position, no head tilt, same minimal neck context, with generous white padding for grid slicing.

Avoid: text, labels, numbers, watermark, hair, hairstyle outline, beard, gendered styling, makeup, color fill, shadows, gradients, background texture, head tilt, changed camera angle, changed face shape, and changed pose.

## Review Notes

Expression assets may include eyes, eyebrows, nose, and mouth because expression needs facial feature cues. They must remain generic enough that the user reads mood rather than gender, hair, styling, or identity changes.
