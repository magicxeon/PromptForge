# Character Sheet Illustrated Set Master Prompt

Create one clean source sheet for a visual character builder.

Style:

- flat vector-like monochrome illustration
- simple charcoal or black line art
- no text, labels, symbols, borders, logos, watermarks, or decorative backgrounds
- pure white background
- consistent scale, line weight, baseline, and padding
- readable as small UI icons after slicing

Subject constraints:

- no detailed face
- no brand logos
- no copyrighted outfit designs
- no sexualized body emphasis
- no medical or judgmental body language
- body and outfit diagrams should communicate selection intent only

Output as one grid sheet. Each cell should be visually centered and fit inside a safe margin.
