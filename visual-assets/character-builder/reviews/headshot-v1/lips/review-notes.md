# Lips Review Notes

## Source

- `visual-assets/character-builder/source-sets/headshot-v1/facial-features/lips/lips-set-r1.png`

## First Pass

The first pass uses only the existing lip-shape attributes:

- `lips.001` Natural lips
- `lips.002` Cherry lips
- `lips.003` Cupid's bow lips
- `lips.004` Plump lips
- `lips.005` Thin lips
- `lips.010` Heart-shaped lips
- `lips.011` Heavy upper lip
- `lips.012` Wide lips

Excluded from this visual set:

- `lips.006` Soft smile
- `lips.007` Closed-mouth smile
- `lips.008` Gentle smile
- `lips.009` Neutral expression

Review focus:

- The icons must remain closed-mouth and expression-neutral.
- Shape differences should be visible at thumbnail size.
- No lipstick color, teeth, skin, smile, or emotion should be introduced.

## Revision Notes

- `r1` was regenerated after review because the first source sheet made the lip silhouettes too similar.
- The current source follows a stronger shape-differentiation reference: compact cherry, sharp cupid's bow, visibly plump, and very thin lips.
- The set was expanded to 8 options with heart-shaped, heavy upper, and wide lips to give the visual builder more anatomy-level variety.
- The set still remains closed-mouth, expression-neutral, and colorless for mask recoloring.
